## NodeJs 入门篇

### 一、编写单元测试 Jest 程序

1. 第一步 `npm install jest -g`
2. `__test__`文件夹中创建`index.spec.js` 文件
3. 文件规则[name].spec.js

```js
test('Hello World', () => {
	require('../index')
})
```

4. 为测试结果添加`断言`实现`测试驱动`

```diff
test('Hello World', () => {
  const result = require('../index')
- expect(result).toBe('Hello Wolrd')
})
```

5. 运行`jest [文件夹名] --watch`动态更新
